Image-video library which displays them in beautiful grids.

# Table of contents
1. [Installation](#installation)
2. [Demo](#demo)
3. [Basic Usage](#basic-usage)

# Installation

- Using NPM
`npm i react-fb-image-video-grid`

- Using Yarn
`yarn add react-fb-image-video-grid`
